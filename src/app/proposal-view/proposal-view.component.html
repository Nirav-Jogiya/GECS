Add a 4th “liquidity contributor” Factor to the weighting algorithm

Expires 3/9/2020 @ 11:23 UTC

Proposed Action

Context

In order to provide more stability and growth to GECS, I believe that we should add more weight to those that support the ecosystem financially.
